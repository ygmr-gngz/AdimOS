# Brand Statik Varlıklar

Bu klasöre şu dosyayı ekle:

```
adim-musavir-logo.png
```

- Format: PNG (şeffaf arka plan)
- Min boyut: 512×512 px
- Dosya adı değiştirilmemelidir — kod `staticFile("brand/adim-musavir-logo.png")` ile okur

Logo dosyası eksikse render sırasında marka köşe logosu görünmez.
